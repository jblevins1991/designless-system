# Breadcrumbs

## Props

The Button's props extend `React.HTMLAttributes<HTMLButtonElement>`. This keeps the button's 
props as close to the JSX specification as possible.

**Props Details**

| Name       | Default Value | Possible Values                      | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| className  |               |                                      | Appends a class to the button element.                       |
| id         |               |                                      | Sets the id on the button element.                           |
| url        |               |                                      | The URI of the web page.                                     |

## Example

<>
    <Breadcrumbs
        url='/category'
    />
</>

```jsx
import { Breadcrumbs } from '@nightowl-developers/styless-react';

const MyComponent = () => {
    return <Breadcrumbs
        url='/category'
    />;
}
```
