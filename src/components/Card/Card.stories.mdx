# Card

## Props

The Button's props extend `React.HTMLAttributes<HTMLDivElement>`. This keeps the card's 
props as close to the HTMLDivElement JSX specification as possible.

**Props Details**

| Name       | Default Value | Possible Values                      | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| children   |               |                                      | The child components to render.                              |
