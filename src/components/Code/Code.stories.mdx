# Code

## Props

React doesn't have natural HTML prop types, so we are taking some 
liberty on the props interface until the next major version.

**Props Details**

| Name       | Default Value | Possible Values                      | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| children   |               |                                      | The code to render.                                          |
| className  |               |                                      | Appends a class to the button element.                       |
| copyIcon   |               |                                      | The icon to render in the copy button.                       |
| id         |               |                                      | Sets the id on the button element.                           |
| language   |               |                                      | The coding language for this particular code block.          |
