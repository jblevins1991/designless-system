# Code

## Props

React doesn't have natural HTML prop types, so we are taking some 
liberty on the props interface until the next major version.

**Props Details**

| Name       | Default Value | Type                                 | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| children   |               | React.ReactNode                      | The child components to render.                              |
| className  |               | string                               | Appends a class to the button element.                       |
| copyIcon   |               | React.ReactNode                      | The icon to render in the copy button.                       |
| id         |               | string                               | Sets the id on the button element.                           |
| language   |               | string                               | The coding language for this particular code block.          |
