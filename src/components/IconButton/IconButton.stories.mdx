# Button

## Props

The Button's props extend `ButtonProps`. This keeps the icon button's 
props as close to the Button component's as possible.

**Props Details**

| Name       | Default Value | Type                                 | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| className  |               |                                      | Appends a class to the button element.                       |
| id         |               |                                      | Sets the id on the button element.                           |
| type       | 'button'      | 'button', 'submit', 'reset'          |                                                              |

## Render Props

Render props are used to change how a component is rendered. Our render props are added to 
a component's className to give developers a way to control how their component works.

**Render Props Details**

| Name       | Default Value | Type                                 | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| color      | 'primary'     | 'primary', 'secondary', 'tertiary'   | Controls the background color used from your design system.  |
| icon       |               | React.ReactNode                      | The icon to render in the button.                            |
| size       | 'medium'      | 'large', 'medium', 'small'           | Controls the size of the button being rendered.              |
| variant    | 'solid'       | 'solid', 'outlined', 'text'          | Controls the visual representation of the button.            |
