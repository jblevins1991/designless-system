# Image

## Props

The Button's props extend `React.HTMLAttributes<HTMLImageElement>`. This keeps the button's 
props as close to the JSX specification as possible.

**Props Details**

| Name       | Default Value | Type                                 | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| alt        |               | string                               | The descriptive text for the image.                          |
| className  |               | string                               | Appends a class to the button element.                       |
| id         |               | string                               | Sets the id on the button element.                           |
| src        |               | string                               | The URL of the image you want to display.                    |

## Example

<>
    <Image
        alt='A picture of puppies.'
        src=''
    />
</>

```jsx
import { Image } from '@nightowl-developers/styless-react';

const MyComponent = () => {
    return <Image
        alt='A picture of puppies.'
        src=''
    />;
}
```
