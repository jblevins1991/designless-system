# Ordered List

## Props

The Button's props extend `React.HTMLAttributes<HTMLOListElement>`. This keeps the list's 
props as close to the HTMLDivElement JSX specification as possible.

**Props Details**

| Name       | Default Value | Type                                 | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| children   |               | React.ReactNode                      | The child components to render.                              |

## Example

<>
    <OrderedList>
        <ListItem>One</ListItem>
        <ListItem>Two</ListItem>
