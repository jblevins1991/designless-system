# Typography

## Props

The Button's props extend `React.HTMLAttributes<HTMLHeadingElement | HTMLParagraphElement>`. This keeps the button's 
props as close to the JSX specification as possible.

**Props Details**

| Name       | Default Value | Possible Values                      | Description                                                  |
|------------|---------------|--------------------------------------|--------------------------------------------------------------|
| className  |               |                                      | Appends a class to the button element.                       |
| id         |               |                                      | Sets the id on the button element.                           |

## Render Props

Render props are used to change how a component is rendered. Our render props are added to 
a component's className to give developers a way to control how their component works.

**Render Props Details**

| Name       | Default Value | Possible Values                         | Description                                                  |
|------------|---------------|-----------------------------------------|--------------------------------------------------------------|
| variant    | 'p'           | 'h1', 'h2', 'h3', 'h4', 'h5', 'h6', 'p' | The visual representation of the text.                       |

## Example

<>
    <Typography variant='h1'>
        Heading element
    </Typography>
    <Typography>
        Paragraph element.
    </Typography>
</>

```jsx
import { Typography } from '@nightowl-developers/styless-react';

const MyComponent = () => {
    return <>
        <Typography variant='h1'>
            Heading element
        </Typography>
        <Typography>
            Paragraph element.
        </Typography>
    </>;
}
```
