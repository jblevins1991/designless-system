# Styling Components

Styling components is simple due to the lack of styles. This will make your 
styles intuitive and have intent. By not having to use more complex specificity 
in your styles, they will be easier to understand and change in meaningful ways.

## Styled-Components

```jsx
import { Button as SButton } from '@nightowl-developers/styless-react';
import styled from 'styled-components';

const Button = styled(SButton)`
  // steal the button styles from the CSS/SCSS section below...
`;

export default Button;
```

## TailwindCSS

boilerplate for all of the components and their states.

## CSS/SCSS

the `TailwindCSS` example and paste that into your CSS file.

starting point.
